### 分布式锁

### 分布式锁的设计原则

- 互斥性，同一时间只有一个线程持有锁 
- 容错性，即使某一个持有锁的线程，异常退出，其他线程仍可获得锁 
- 隔离性，线程只能解自己的锁，不能解其他线程的锁 

 ### 系统稳定性

#### 日志监控

查看请求访问量

```bash
cat access.log | cut -f1 -d " " | sort |uniq -c | sort -k 1 -n -r | head -10
```

查看最耗时页面

```bash
cat access.log | sort -k 2 -n -r | head -10
```

#### 系统监控

查看内存使用多的进程

```bash
ps aux | sort -k4nr | head -n 10
```

查看cpu使用多的进程

```bash
ps aux | sort -k3nr | head -n 10
```
